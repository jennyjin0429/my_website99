Another take away from this assignment is how to adjust the axis so the plot will look more clear to the audience. For example, for the bar graphs, we need to think carefully about which variables should go on the y-axis. The last takeaway is we should reorder the data on the y-axis to make the plot look nicer. I will use this plot as an example.

top_25_beer_servings <- drinks %>% 
  arrange(desc(beer_servings)) %>% 
  slice(1:25)

ggplot(top_25_beer_servings, 
       aes(x = beer_servings, 
           y = reorder(country,beer_servings)))+
  geom_bar(stat="identity")+
  labs(title = "Which country drinks the most beer?", 
       subtitle = "Top 25 beer consuming countries",
       caption="fivethirtyeight", x="Beer Servings",y="Country")+
  theme_economist()+
  NULL

If you look at this chart, it will be less appealing if you put the country on x-axis as there will be too many countries crowded on the x-axis.
